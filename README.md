# ImagePicker
Ionic2+AngularJS2 图片选择component和多图上传

* 功能如下:
    * 1.选择照相机或者相册里的图片并且展示
    * 2.多张图片异步上传至服务器

* 依赖插件:
    * $ ionic plugin add https://github.com/Telerik-Verified-Plugins/ImagePicker
    * $ ionic plugin add cordova-plugin-camera
